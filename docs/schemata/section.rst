Sections
========

Sections are collections of questions which are grouped together on the same block (page) of a questionnaire.

.. code-block:: javascript

    {
        "description": "",
        "display": {
            "properties": {}
        },
        "id": "ed3e200a-0735-4e8d-9eea-627c1d908697",
        "questions": [
          ...,
          ...
        ],
        "title": "Choose your side",
        "validation": []
    }
